📖 Integration Guide

Basic Integration

Add these two lines to your HTML page to integrate the chatbot widget:

<link rel="stylesheet" href="/css/chatbot-widget.css"> <script src="/js/chatbot-widget.js"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot', position: 'bottom-right', theme: 'light' }); </script>

Custom Configuration

Customize the widget appearance and behavior:

ChatBotWidget.init({ apiEndpoint: '/api/chatbot', position: 'bottom-right', theme: 'light', title: 'Chat Assistant', subtitle: 'How can we help?', placeholder: 'Type your message...', colors: { primary: '#007bff', secondary: '#6c757d', background: '#ffffff', text: '#333333', botMessage: '#f0f0f0', userMessage: '#007bff' } });

ASP.NET Core Integration

Add to your shared layout file (_Layout.cshtml):

<!-- In the <head> section --> <link rel="stylesheet" href="~/css/chatbot-widget.css"> <!-- Before closing </body> tag --> <script src="~/js/chatbot-widget.js"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot' }); </script>

ASP.NET MVC Integration

Add to your master page (Site.Master):

<!-- In the <head> section --> <link rel="stylesheet" href="<%= ResolveUrl("~/css/chatbot-widget.css") %>"> <!-- Before closing </body> tag --> <script src="<%= ResolveUrl("~/js/chatbot-widget.js") %>"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot' }); </script>

ASP.NET WebForms Integration

Add to your master page (.Master):

<!-- In the <head> section --> <link rel="stylesheet" href="<%= ResolveUrl("~/css/chatbot-widget.css") %>" /> <!-- Before closing </body> tag --> <script src="<%= ResolveUrl("~/js/chatbot-widget.js") %>"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot' }); </script>

🔌 API Endpoints

POST /api/chatbot/send

Send a message to the chatbot and receive a response.

Request Body: { "content": "Hello, how are you?", "messageType": "text", "sender": "user" } Response: { "responseId": "uuid", "content": "Hello! I'm here to help.", "messageType": "text", "status": "success", "sender": "bot", "suggestedResponses": [...], "showTypingIndicator": false }

GET /api/chatbot/suggestions

Get initial suggested responses.

Response: { "content": "What can I help you with?", "suggestedResponses": [ { "text": "View Appointments", "payload": "view_appointments", "actionType": "button" }, ... ] }

POST /api/chatbot/action

Handle action-based responses from suggested buttons.

Request Body: { "payload": "view_appointments" } Response: { "content": "Loading your appointments...", "status": "success", "metadata": { "action": "redirect", "target": "/Family/AppointmentSchedule" } }

GET /api/chatbot/health

Health check endpoint.

Response: { "status": "healthy", "timestamp": "2024-01-15T10:30:00Z", "service": "ChatBot API" }
