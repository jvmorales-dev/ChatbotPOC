📖 Integration Guide

Basic Integration

To integrate the chatbot widget into any HTML page, simply add one script tag:

<script src="/Scripts/Chat/chatbot-widget-complete.js"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot', position: 'bottom-right', theme: 'light' }); </script>

Advanced Configuration

Customize the widget with additional options:

ChatBotWidget.init({ apiEndpoint: '/api/chatbot', position: 'bottom-right', theme: 'light', title: 'Chat Assistant', subtitle: 'How can we help?', placeholder: 'Type your message...', colors: { primary: '#007bff', secondary: '#6c757d', background: '#ffffff', text: '#333333', botMessage: '#f0f0f0', userMessage: '#007bff' } });

ASP.NET WebForms Integration

Add to your master page (.Master):

<!-- Before closing </body> tag --> <script src="<%= ResolveUrl("~/Scripts/Chat/chatbot-widget-complete.js") %>"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot' }); </script>

ASP.NET MVC Integration

Add to your layout file (_Layout.cshtml):

<script src="~/Scripts/Chat/chatbot-widget-complete.js"></script> <script> ChatBotWidget.init({ apiEndpoint: '@Url.Action("Index", "ChatBot")' }); </script>

ASP.NET Core Integration

Add to your layout file (_Layout.cshtml):

<script src="~/js/chat/chatbot-widget-complete.js"></script> <script> ChatBotWidget.init({ apiEndpoint: '/api/chatbot' }); </script>

🔌 API Methods

Public Methods

// Open the widget ChatBotWidget.open(); // Close the widget ChatBotWidget.close(); // Toggle widget open/closed ChatBotWidget.toggle(); // Clear all messages ChatBotWidget.clearMessages(); // Destroy and remove the widget ChatBotWidget.destroy(); // Re-initialize with new config ChatBotWidget.init({ ... });

Expected API Endpoints

Your backend should implement these endpoints:

POST /api/chatbot/send

Send a message to the chatbot and receive a response.

Request: { "content": "Hello, how are you?", "messageType": "text", "sender": "user" } Response: { "content": "Hello! I'm here to help.", "messageType": "text", "sender": "bot", "suggestedResponses": [ { "text": "View Appointments", "payload": "view_appointments", "actionType": "button" } ] }

GET /api/chatbot/suggestions

Get initial suggested responses for the widget.

Response: { "content": "What can I help you with?", "suggestedResponses": [...] }

POST /api/chatbot/action

Handle action-based responses from suggested buttons.

Request: { "payload": "view_appointments" } Response: { "content": "Loading your appointments...", "suggestedResponses": [...], "metadata": { "action": "redirect", "target": "/appointments" } }
